# 更新日志

## [1.1.0] - 2026-05-29

### 新增

- **循环待办**：支持每天/每周/每月重复提醒，可自定义具体时间和目标日
  - 每天：选择具体几点提醒
  - 每周：选择星期几 + 几点提醒
  - 每月：选择几号 + 几点提醒
  - 到达设定时间后自动进入下一轮，无需手动操作
  - 循环待办不可标记完成，只能删除
- **拖拽排序**：长按待办条目拖动即可调整顺序，5px 移动阈值避免误触
- **标题栏版本号**：左上角动态显示当前版本号

---

## [1.0.0] - 2026-05-23

### 功能

- 本地待办：SQLite 存储，支持新增/编辑/删除/完成
- 置顶、重要程度颜色标记
- 截止时间：快速设置（今天/明天/本周末）和自定义日期时间
- 截止提醒：到期前 15 分钟系统通知（Rust 后端 + 前端双保险轮询）
- 时钟与农历：可选显示（默认折叠）
- 无边框透明面板，毛玻璃效果，透明度可调
- 窗口置顶、关闭隐藏到系统托盘
- 全局快捷键 `Ctrl+Shift+L` / `⌘+Shift+L` 唤出/隐藏
- 独立设置页：透明度、时钟折叠、语言（中/英/跟随系统）、开机启动
- 跨平台：Windows（NSIS + 便携版）、macOS（.app / .dmg）
