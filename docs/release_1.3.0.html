v1.3.0 · 本次重点

三大核心更新

导出 TXT

底部一键导出全部待办为纯文本，未完成 / 已完成分区整理，方便备份、转发与归档。

专注 / 完整模式

完整模式管全部待办；专注模式只显示未完成项，窗口自动变矮，真正的小部件体验。

弹窗强提醒

到期弹出独立提醒窗，置顶显示、不易错过；介意打断可在设置里切换为系统通知。

v1.3.0 · 导出 TXT

待办一键导出，随时备份分享

完整模式底部点击「导出」，选择保存路径，生成结构化 TXT 文件。

导出内容

  • 未完成区：内容 + 截止时间（如有）
  • 已完成区：内容 + 完成时间
  • 置顶项优先排列
  • 纯文本格式，任意编辑器可打开

适用场景

  • 周报 / 日报前快速整理任务清单
  • 发给同事或备份到网盘
  • 换机前本地数据归档
  • 导出成功后有 Toast 提示
点击导出底部按钮
→
选择路径系统保存对话框
→
生成 TXT分区写入
→
导出成功界面提示
v1.3.0 · 专注 / 完整模式

两种模式，一张桌面小窗

🎯 专注模式

  • 仅未完成待办
  • 勾选完成，只读列表
  • 窗口自动变矮（最多 5 条 + 滚动）
  • 隐藏顶栏、日历、底部操作栏
⇄

📋 完整模式

  • 周日历 · 时钟 · 全部待办
  • 添加 / 编辑 / 拖拽 / 导出
  • 设置 · 主题 · 提醒方式
  • 退出专注后恢复原先窗口大小

切换方式：Ctrl/Cmd + Shift + F  或  托盘右键菜单

托盘菜单：显示窗口 · 专注/完整 · 窗口置顶 · 退出

v1.3.0 · 弹窗提醒

强提醒，到期不会漏

截止前 15 分钟触发提醒。默认弹出独立置顶小窗，比系统通知更醒目。

弹窗提醒（默认）

  • 独立提醒窗口，始终置顶
  • 显示待办内容与剩余 / 逾期时间
  • 强提醒：会打断当前打字焦点
  • 支持「我知道了」「稍后提醒」

系统通知（可选）

  • 设置 → 提醒方式 → 系统通知
  • 走操作系统通知中心，不抢焦点
  • 适合不希望被打断的场景
  • 后台 Rust + 前端双保险轮询
v1.3.0 · 更多更新

体验细节，一并到位

操作提示

  • 删除、导出、清除已完成等操作有 Toast 反馈
  • 成功 / 失败信息底部居中显示
  • 不用猜「到底有没有生效」

记住窗口位置与大小

  • 关闭 / 隐藏时自动保存窗口状态
  • 下次打开回到上次的位置和尺寸
  • 专注模式隐藏时不污染完整模式尺寸

纯托盘小部件

  • Windows 不在任务栏显示
  • macOS 不进 Dock，仅托盘图标
  • 需要时唤出，平时不占桌面任务栏位

快捷键一览

  • Ctrl/Cmd + Shift + L 显示 / 隐藏窗口
  • Ctrl/Cmd + Shift + F 专注 / 完整模式
  • Ctrl/Cmd + Shift + P 窗口置顶
